## ðŸŽ¯ Common Tasks

### Check if a Store Needs Stock
1. Go to **Stock Status** tab
2. Find the store (or filter by store name)
3. Look for red/orange products
4. Note quantities needed

### Respond to Critical Alerts
1. Go to **Alerts** tab
2. Click **High** filter
3. Read alert details
4. Take recommended action

### Create a Stock Request (Store Incharge)
1. Go to **Requests** tab
2. Click **New Request**
3. Select priority level
4. Enter quantities needed
5. Add notes (optional)
6. Submit

### Fulfill a Request (Production Head)
1. Go to **Requests** tab
2. Switch to **Pending** tab
3. Click **Process Request**
4. Adjust quantities (if needed)
5. Add fulfillment notes
6. Submit

### Generate Monthly Report
1. Go to **Reports** tab
2. Select report type
3. Choose date range (30 days)
4. Review data
5. Click **Export Report**

### Plan Next Week's Production
1. Go to **Forecast** tab
2. Review 7-day forecast
3. Check stock-out risks
4. Note recommended production
5. Adjust schedule accordingly

### Compare Store Performance
1. Go to **Rankings** tab
2. Select timeframe
3. View top 3 podium
4. Scroll to detailed table
5. Analyze metrics

## ðŸ†˜ Troubleshooting

### "I don't see Inventory Analytics button"
**Solution:** Check your role. This feature is only for Operations Managers and Cluster Heads.

### "No data is showing"
**Solution:** 
1. Ensure you have data in the system (sales, production, requests)
2. Refresh the page
3. Check your internet connection

### "Charts are not loading"
**Solution:**
1. Refresh the page
2. Clear browser cache
3. Try a different browser

### "Calculations seem wrong"
**Solution:** Remember that stock = fulfilled requests - estimated sales. If you have no fulfilled requests, stock will show as 0.

## ðŸ“± Best Practices

### 1. Check Alerts Daily
Set a routine to check alerts every morning. Address high-priority alerts immediately.

### 2. Use Priority Levels Correctly
- **Urgent:** Store has < 1 day stock
- **High:** Store has < 3 days stock
- **Medium:** Regular restock
- **Low:** Preventive ordering

### 3. Trust the Forecast
The 7-day forecast uses historical data. Use it to plan ahead and avoid rush orders.

### 4. Monitor Rankings Weekly
Track store performance trends. Investigate sudden drops in rankings.

### 5. Export Reports Monthly
Keep monthly reports for historical comparison and business analysis.
